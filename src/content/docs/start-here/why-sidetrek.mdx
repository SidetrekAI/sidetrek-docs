# Why Sidetrek?
Sidetrek is a tool to help you create and manage your content. It's a static site generator that uses Astro, a new static site generator that uses React components to build your site. Sidetrek is designed to be easy to use and flexible, so you can create the site you want without having to learn a lot of new tools.
