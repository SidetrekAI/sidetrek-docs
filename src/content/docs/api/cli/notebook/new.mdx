# sidetrek notebook new

The `sidetrek notebook new` command is used to create a new Jupyter notebook.

There are two options available for creating a new notebook:

1. **Python** - Create a new blank Python notebook.
2. **DBT** - Create a new DBT notebook. This creates a new Jupyter notebook with the DBT code template.
