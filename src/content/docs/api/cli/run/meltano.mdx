# sidetrek run meltano *

The `sidetrek run dbt *` command allows you to execute Meltano commands within your project virtual environment and in the right directory (Meltano project root). 

## What's Happening Underneath?

This command provides a convenient way to run Meltano CLI commands without having to move into Meltano project directory and activating virtual environment.

It's identical to running:

<BashCode code="cd your_project/meltano && poetry run meltano *" client:load />

## Usage

<BashCode code="sidetrek run meltano <any meltano command and options>" client:load />

## Examples

### Check Meltano Version

This command outputs the current version of Meltano installed in your Sidetrek environment.

<BashCode code="sidetrek run meltano --version" client:load />

### Add a New Extractor

This command adds the `tap-github` extractor to your Meltano project.

<BashCode code="sidetrek run meltano add extractor tap-github" client:load />
