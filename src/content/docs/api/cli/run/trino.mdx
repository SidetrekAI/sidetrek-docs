# sidetrek run trino shell

The `sidetrek run trino shell` command puts you in the Trino shell, allowing you to interactively run SQL queries against your local Trino cluster. 

It's convenient when you want to directly query Iceberg tables for debugging or ad hoc queries, for example.

## Usage

<BashCode code="sidetrek run trino shell" client:load />

## What's Happening Underneath?

It's identical to running:

<BashCode code="docker exec -it trino trino" client:load />

## Examples

### Enter Trino Shell

<BashCode code="sidetrek run trino shell" client:load />

This command opens the Trino command-line interface.

### Run a SQL Query:

Once in the Trino CLI, you can execute SQL queries.

For example, this command retrieves all the catalogs in trino database.

<BashCode code="show catalogs;" client:load />

### Exit Trino Shell

To exit the Trino CLI, simply type:

<BashCode code="exit;" client:load />

Or:

<BashCode code="quit;" client:load />
