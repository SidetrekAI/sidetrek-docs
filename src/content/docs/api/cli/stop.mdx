# sidetrek stop

The `sidetrek stop` command stops all running docker containers.

## Usage

<BashCode code="sidetrek stop" client:load />

## What's Happening Underneath?

This command stops all running docker containers including Minio, Iceberg, Trino, and Superset.

It's simply running:

<BashCode code="docker compose stop" client:load />

And:

<BashCode code="cd superset && docker compose stop" client:load />
