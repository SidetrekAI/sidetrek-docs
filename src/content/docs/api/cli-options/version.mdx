# --version

The `--version` option displays the current version of Sidetrek installed on your system. This is useful for verifying the installed version, especially when troubleshooting or ensuring compatibility with other tools and projects.

### Usage
<BashCode code="sidetrek --version" client:load />
When you run the above command, it outputs the version of Sidetrek you have installed, in this case, version 0.1.34
```plaintext
0.1.34
```
